# Actividad 6: Multilayer neural network with hyperparameter optimisation

Esta actividad actividad en realizar un modelo de red neuronal multicapa aplicado a un problema real de clasificación multiclase utilizando una base de datos de Kaggle sobre los indicadores de salud del diabetes. Primero, se selecciona y analiza un conjunto de datos, realizando limpieza y análisis exploratorio para comprender sus características. Posteriormente, se diseña, entrena y ajusta una red neuronal, incorporando técnicas como la estandarización de datos, la optimización de hiperparámetros y el uso de early stopping para evitar el sobreajuste. Además, se evalúa el desempeño del modelo mediante métricas como precisión, recall y matriz de confusión, así como el análisis de curvas de aprendizaje para diagnosticar su comportamiento. Finalmente, se guarda el modelo entrenado y se reflexiona sobre los resultados obtenidos, los retos enfrentados y los aprendizajes adquiridos durante todo el proceso.
